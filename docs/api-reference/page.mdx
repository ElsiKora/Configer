# API Reference

This section is split into focused pages for faster lookup and long-term maintenance.

## Reference Index

- **Factories** - `createConfiger` and `createConfigerSync`
- **Interfaces** - client/options/result/plugin/schema contracts
- **Types** - loader/search/watch/config types
- **Error Codes** - stable runtime `ConfigError.CODE` values

## Fast Navigation

- Looking for constructor input? -> Interfaces (`IConfigOptions`)
- Looking for callback/loader signatures? -> Types
- Looking for fail-fast semantics? -> Error Codes
