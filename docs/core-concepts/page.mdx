# Core Concepts

This section documents how Configer resolves, validates, merges, caches, and watches configuration.

## Concept Map

- **Resolution Flow** - how search plan is built and executed.
- **Inheritance and Merge** - precedence, source tracking, cycle protection.
- **Environment Overrides** - `$env` and `$<name>` semantics.
- **Schema Validation** - descriptor contract and issue model.
- **Caching and Watch** - cache layers, watch lifecycle, callback behavior.

1. Resolution Flow
2. Inheritance and Merge
3. Environment Overrides
4. Schema Validation
5. Caching and Watch
